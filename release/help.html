基本用法

2024年开始，知乎网站加入反爬虫机制，所以不能用 curl 下载知乎网页的内容。

如果在知乎网上看到一个回答很好，想保存下来，怎么办？

比如：

中信建投发布研报称「中国房价有望在 2026 年前后止跌回稳」，怎样看待这一预测

如果现在想保存这个回答，应该如何做？


本工具就可以实现这个功能

1. 先用浏览器打开这个链接，然后保存这个网页

Windows Edge 浏览器： ctrl + s， 就可以保存当前网页

这里假设保存的文件名是： question.html


2. 运行程序 zhihu_crawler.exe；

选择刚才保存的文件，例如 question.html


3. 点击生成网页，就会在当前目录的 temp 子目录（程序自动创建的）下，

看到生成的文件 out.html

这个文件的内容可以随便拷贝，例如拷贝到邮件中，发给自己，保存下来

拷贝网页内容时，包含图片

需要安装 python

生成网页后，双击运行 start.bat 启动http 服务（和zhihu_crawler.exe 在同一目录下）

启动 http 服务器后

浏览器端： 在浏览器中输入 访问生成的知乎问答网页

然后就可以拷贝文字和图片内容

ctrl a, ctrl c, ctrl v
